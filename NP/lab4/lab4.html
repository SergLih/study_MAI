Разработать 2-3 примера с использованием JQuery, который включал бы функции из разделов

Раздел Функция JQuery Описание Пример
Атрибуты hasClass() Проверяет наличие класса cName хотя бы у одного из выбранных элементов
CSS css() Возвращает или изменяет значения css-величин у выбранных элементов страницы
Селекторы (фильтры элементов формы) :reset Соответствует кнопкам, которые сбрасывают введенные в формы данные: input-элементы с типом reset.
Манипуляции, работа с параметрами и стилями offset() Изменяет координаты элемента
Манипуляции, обновление/удаление содержимого empty() Удаляет все содержимое у выбранных элементов (без удаления самих элементов)
Работа с набором, перемещение по DOM prev() Для каждого из выбранных элементов находит предшествующий ему элемент (но только если он лежит на том же уровне иерархии DOM)
События базовые jQuery.proxy() По заданной функции, создает другую такую же, внутри которой переменная this будет равна заданному значению
События мыши dblclick() Устанавливает обработчик двойного "клика" мышью по элементу, либо, запускает это событие
События клавиатуры и формы focus() Устанавливает обработчик получения фокуса, либо, запускает это событие
Эффекты fadeToggle() Вызов этого метода приводит к плавному исчезновению (если элемент не скрыт) или появлению (если элемент скрыт) выбранных элементов на странице, за счет изменения прозрачности
